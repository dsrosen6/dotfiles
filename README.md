## Steps to install
1. Make sure you have `homebrew` installed.
2. Install oh-my-zsh: `sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"`
3. Run this command in terminal: `bash -c "$(curl -fsSL https://raw.githubusercontent.com/dsrosen6/dotfiles/main/setup.sh)`
4. If you'd like to use my iTerm settings, go to iTerm's Settings > General > Preferences, check "Load preferences from a custom folder or URL", then pick the .iterm2 folder in this repo.
