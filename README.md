## Steps to install
1. Make sure you have `homebrew` installed.
2. Install oh-my-zsh: `sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"`
3. Clone this repo to your home directory.
4. Make `auto-setup.sh` executable by running `chmod +x dotfiles/auto-setup.sh`
5. Run the `auto-setup.sh` script. Then, quit and open your terminal to see changes.
6. If you'd like to use my iTerm settings, go to iTerm's Settings > General > Preferences, check "Load preferences from a custom folder or URL", then pick the `.iterm2` folder from this repo.
